# 🌍 Tradutor Inteligente de Arquivos

## 📝 Sobre o Projeto
Tradutor automático de arquivos que detecta o idioma de origem e traduz para o destino escolhido. Suporta múltiplos formatos!

## ✨ Funcionalidades
- 📄 Suporte a: TXT, PDF, DOCX, CSV
- 🌐 Detecção automática de idioma
- 🔄 Tradução para 8+ idiomas
- 💾 Download do arquivo traduzido
- 🎨 Interface bonita e intuitiva

## 🛠️ Tecnologias
- Python
- Streamlit
- Google Translate API
- PyPDF2, python-docx, pandas

## 🚀 Como usar
1. Faça upload do arquivo
2. Escolha o idioma de destino
3. Clique em "Traduzir agora"
4. Baixe o resultado

## 📦 Instalação

pip install -r requirements.txt
